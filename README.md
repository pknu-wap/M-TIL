## 🔫 규칙

### 📁 주차별 파일에 목표를 세우고 그 주에 배운 것들을 작성 📁
- 알고리즘, 개발, 독서등을 통해서 배운것들을 작성해주세요
- KPT회고 방식에 맞춰서 작성해주세요 [KPT회고란?](https://brunch.co.kr/@jinha0802/35)
- 개인별 README.md에 주차, 날짜, 주제, 링크를 추가해 주세요
  - 예시

| 주차 | 날짜 | 주제 | 링크 |
|--|--|--|--|
| 1주차 | 24.09.16-24.09.22 | Kotlin의 Data Class | [블로그](https://blog.naver.com/pluto0303/223394632193) , [github](www.github.com/pknu-wap/M-TIL)

### ☕ 공부 외로 운동, 독서, 먹은 커피수 등 기록도 가능! ☕
- 목표와 연관된 어떤 내용도 작성해도 좋습니다
- 개인 README나 파일을 만들어서 작성해주세요 

### ✔️ 군생활 또는 몇개월 동안 달성할 장기 목표 세우기 ✔️
- 각자의 이름으로 되어있는 폴더의 README.md에 목표를 작성합니다
